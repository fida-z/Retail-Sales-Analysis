# Retail-Sales-Analysis
This project works to perform end-to-end data cleaning and exploratory data analysis (EDA) on a relational, raw dataset using SQL. The main goal is to clean and standardise the data, create relational integrity using appropriate primary and foreign keys, perform EDA to uncover business insights and prepare the raw dataset for further dashboard creation and report generation.

The dataset used is an artifically generated dataset that simulates real-world retail working, containing tables about customers, products, stores, orders and returns. The dataset contains inconsistencies, missing values, duplicates and formatting problems.

The Schema consists of these tables :
- Customers
- Products
- Stores
- Orders
- Order_Items
- Returns
